# 💼 HR Analytics Dashboard  

An interactive **HR Analytics Dashboard** built in **Microsoft Excel** to track and visualize key workforce insights.  
It helps HR teams monitor employee data, analyze attrition trends, and make **data-driven decisions** to improve retention and performance.  

---

## 🌟 Key Highlights  
- 📊 **Comprehensive HR Overview:** Total employees, active staff, attrition rate, and average age  
- 🧩 **Attrition Analysis:** Breakdown by department, education, age group, and job role  
- 👩‍💼 **Diversity Insights:** Gender distribution and education-level analysis  
- 😊 **Job Satisfaction Tracker:** Understand employee engagement across roles  
- 🎯 **Interactive Filters:** Explore data dynamically for deeper insights  

---

## 🧠 Skills & Tools Used  
- **Microsoft Excel**  
- **Pivot Tables & Charts**  
- **Conditional Formatting**  
- **HR Analytics & Data Interpretation**  

---

## 📈 Dashboard Insights  
This dashboard empowers HR leaders to:  
✅ Identify key factors behind employee attrition  
✅ Monitor demographic patterns across the workforce  
✅ Support strategic HR planning and retention strategies
